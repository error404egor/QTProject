Приложение "Умный олодильник"



Содержание.
    1 Концептуальная модель.
    2 Функции.
    3 Путь пользователя.
        3.1 User case.
        3.2 User flow
    4 Используемые API.
    5 Пользовательский интерфейс.


1 Концептуальная модель.

Проет будет иметацией работы умного холодильника и его функций. Направлен он будет на
людей, готовых проинвестировать деньги на создание настоящего умного холодильника. 

2 Функции.

        -Распознование продуктов по картинке
        -Поиск по холодильнику
        -Автосоздание корзины покупок

3 Путь пользователя.

3.1 User case.
    Программа будет применяться непосредственно в умном холодильнике а также будет 
    служить деммонстрацией его работы.
3.2 User flow
    Будет возможность выбрать папку фотографиями
    кнопка обновления содержимого
    отображение товаров, которые сейчас в холодильнике
    отображение списка покупок
    список товаров, которые "всегда должны быть в холодильнике"

4 Используемые API.
    PyQt5
    PIL
    sqlite3
    imagehash

5 Пользовательский интерфейс.
    Простой пользовательский интерфейс на базе QtDesigner'а
